About The Event

The Machine Learning Accelerator Summit, a two-day event hosted by the IEEE Jadavpur University Student Branch. The event focuses on laying the foundations of Machine Learning with hands-on workshops, tech talks, and fundamental insights by exploring advanced concepts from scratch and implementing them in live projects.
